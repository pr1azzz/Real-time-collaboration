# Task Manager - Канбан-доска с real-time обновлениями

## Описание

Веб-приложение для управления задачами в реальном времени с Kanban-доской, поддержкой ролей (владелец, редактор, просмотр) и возможностью отмены изменений (undo/redo). Приложение позволяет создавать проекты, управлять задачами и работать в команде с мгновенным обновлением изменений через WebSocket.

### Основные возможности

- **Аутентификация и авторизация** (JWT + RBAC)
  - Роли: Владелец (owner), Редактор (editor), Просмотр (viewer)
  - Владелец может управлять участниками и их ролями

- **Управление проектами**
  - Создание, редактирование и удаление проектов
  - Добавление участников в проект

- **Kanban-доска** с 3 колонками
  - Новые → В работе → Готово
  - Drag-and-drop перемещение задач
  - Real-time обновление через WebSocket

- **Управление задачами**
  - Создание, редактирование, удаление задач
  - Drag-and-drop перемещение между колонками
  - История изменений (undo/redo)

- **Real-time коллаборация**
  - Мгновенное отображение изменений у всех участников
  - Уведомления о действиях других пользователей

- **История изменений**
  - Отслеживание всех действий с задачами
  - Возможность отмены последнего действия (Undo)

## Стек технологий

### Frontend
- **React 18** - библиотека для построения пользовательских интерфейсов
- **Vite** - быстрый сборщик и dev-сервер
- **React Router DOM** - маршрутизация
- **Axios** - HTTP-клиент для API запросов
- **Socket.IO-client** - WebSocket клиент
- **React Beautiful DnD** - Drag-and-drop функциональность

### Backend
- **Node.js 18** - среда выполнения JavaScript
- **Express** - веб-фреймворк для API
- **Socket.IO** - WebSocket сервер для real-time обновлений
- **JWT** - JSON Web Token для аутентификации
- **bcryptjs** - хеширование паролей

### База данных
- **PostgreSQL 15** - реляционная база данных
- **pg** - драйвер для Node.js

### Инфраструктура
- **Docker** - контейнеризация приложения
- **Docker Compose** - оркестрация сервисов
- **pgAdmin4** - веб-интерфейс для управления БД
